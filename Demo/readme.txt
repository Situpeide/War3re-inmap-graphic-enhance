War3重制版画面增强Mod v1.01
War3GraphicEhance Mod v1.01

-------------------------------------------------------------------------

安装：

使用.msi安装文件即可。
注意一定要安装在你的魔兽争霸目录。
一般来说位于C:\Program Files\Warcraft III
安装之后需要单独下载地图。
下载后请将地图文件放进 - 我的文档\Warcraft III\Maps\(自己新建任意文件夹)

你可以在官网，Github，Bilibili，知乎找到下载地址以及相关信息。

官网  -  http://www.warchasersreforged.com/   
Bilibi  -  https://space.bilibili.com/21723609
知乎  -  https://zhuanlan.zhihu.com/IMAGINEDYNASTY  
Github  - https://github.com/UIZorrot/War3re-inmap-graphic-enhance

-------------------------------------------------------------------------

概述：

这个Mod目前能够替换部分地形和装饰物纹理

- 洛丹伦的冬天
- 洛丹伦的夏天
- 村庄
- 城邦 
- 灰谷 
- 诺森德

大部分地形的光线，水面都能替换。
这个Mod可以配合其他插件使用。
例如：Gshade，英伟达滤镜

-------------------------------------------------------------------------

注意：

这个Mod不能替换悬崖文件（因为casc文件检索导致）
你可以用这个Mod游玩任何地图，但是会出现悬崖和地面纹理割裂的情况

你必须手动解决这个问题才能使画面效果达到最佳
这个方法是进入你的  -  魔兽根目录\replaceabletextures\cliff\ 
这个文件夹里包括了所有地形纹理的替换方案。

例如：
你在游玩人族战役第一关时发现了问题，本关地形为洛丹伦的夏天。
你应该在Cliff文件夹当中选择 - cliff\洛丹伦的夏天
将其中的所有文件复制到 cliff\ 文件夹目录
就可以解决这个问题。

-------------------------------------------------------------------------

配套地图：

这个Mod在搭配对应地图时可以达到最佳效果。
地图中的脚本无法打包进Mod（因为会引起掉线）
所以只能独立安装
将地图文件放进 - 我的文档\Warcraft III\Maps\(自己新建任意文件夹) 即可使用

脚本内容包括：

- 对应悬崖纹理
- 对应装饰纹理
- 照明效果切换
- 自定义画面选项
- 模拟植被
- 模拟遮蔽
- 昼夜变换
- 嘉年华UI
- 辨识（未完成）
- 天气变换
- 光源
